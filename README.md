## melon top100 가사 크롤러

### 사용법

윈도우, 맥에 따라 crawler.py -> initDrive 함수 내 Chrome Driver 경로 변경해줄 것.

### 2022년 TOP100

`python crawler.py`

#### output

Lyrics_top100.csv

### 2011 ~ 2021년 TOP50

`python crawler_year.py`

#### output

Lyrics_top50_2021.csv, Lyrics_top50_2020.csv, ..., Lyrics_top50_2011.csv
